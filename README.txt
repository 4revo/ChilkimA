Chilki mA - 局域网文件传输工具

使用说明：
1. 确保已安装 Java 8 或更高版本
   - 可以从 https://www.java.com 下载安装
   - 安装完成后重启电脑

2. 运行程序：
   - 双击 run.bat 文件启动程序
   - 程序会自动打开浏览器访问 http://localhost:8848
   - 如果浏览器没有自动打开，请手动访问上述地址

3. 使用功能：
   - 在左侧输入用户名并点击"进入"
   - 可以查看在线用户列表
   - 点击用户头像可以发送文件
   - 接收到的文件会保存在程序所在目录的 user 文件夹中

4. 关闭程序：
   - 在命令行窗口按 Ctrl+C
   - 或直接关闭命令行窗口

注意事项：
- 确保防火墙允许 Java 程序访问网络
- 如果需要更改端口号，请编辑 run.bat 文件
- 程序默认端口为 8848
